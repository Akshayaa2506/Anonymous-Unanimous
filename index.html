ANONYMOUS UNANIMOUS

Anonymous Unanimous enables individuals in a community to seek help from their neighbors via SMS. Initially, the individual scans a QR code that directs them to this website, where they can enter their name, mobile number, and location. These details are then stored in a database. When the individual needs assistance, the individual can send message to the specified number on website and the middleman server will forward the message to all registered users in the same location, ensuring that help reaches those in need efficiently.

OUR SERVICES

1. User Registration via QR Code:
* Individuals can scan a QR code to access the registration website.
* They enter their name, phone number, and location, which is stored in a database.
2. SMS-Based Help Request:
* Users can send an SMS to a specific number when they need assistance.
* No internet is required, making it accessible in remote areas.
3. No installation of any app
4. Location-Based Filtering:
* Messages are sent only to registered users in the same location, ensuring relevant help.
5. Profanity Filtering:
* The system checks messages for inappropriate language before forwarding them.

HOW IT WORKS

Step 1: Register using your name and mobile number.
Step 2: A specified number will be generated for you.
Step 3: Save this specified number on your mobile.
Step 4: Use this number to communicate to us via messages.
Step 5: Allow location permissions for an optimized experience.
